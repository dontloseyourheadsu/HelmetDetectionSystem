Sterile-Field Micro-Contaminant Detector

Automated Detection of Hair and Debris using Hybrid Computer Vision

Department of Computer Science & Engineering, UDLAP
1. Introduction & Problem

The integrity of sterile fields is critical to preventing Surgical Site Infections (SSI). Manual verification is prone to human error due to the microscopic nature of contaminants.


Objective: Detect hair and debris on textured surgical cloths using non-invasive Computer Vision.


Key Challenges:

  • Low contrast (dark hair on dark shadow).
  • Cloth folds mimic contaminant shapes.
2. Preprocessing Pipeline

We apply a Morphological Top-Hat Transform to separate small objects (high frequency) from the background lighting (low frequency).

Fig 1. Data Flow: The Top-Hat operation highlights elements smaller than the structural element (9x9 kernel).
3. Hybrid Model Architecture

A. CNN Classifier (Supervised)

A custom VGG-style network trained to classify specific types of trash.

Fig 2. CNN Architecture: Sequential convolution blocks followed by a dense classification head.

B. Roboflow / SAM (Zero-Shot)

The Segment Anything Model uses a heavy encoder for the image and a light encoder for prompts.

Fig 3. SAM: Prompts ("Find lines") allow real-time queries on top of the pre-computed image embedding.

C. Autoencoder (Unsupervised)

Anomaly detection model. It learns to reconstruct the clean cloth. Any hair causes a high reconstruction error.

Fig 4. Autoencoder: High MSE (Reconstruction Error) indicates contamination.

Clarification on Best Model:

  • CNN (98%): Excellent performance in the lab, but fails if a new type of trash appears that it wasn't trained on.
  • Autoencoder (81%): The preferred solution. It detects anything that isn't cloth (anomalies), making it safer for patients.
  • SAM (60%): Struggles with cloth folds.
5. Conclusion

1. The "Texture" Solution: We conclude that unsupervised learning (Autoencoder) is superior for this medical application. It treats the problem as "Texture Anomaly Detection" rather than "Object Detection," bypassing the need to label thousands of trash examples.

2. Preprocessing is Vital: The Top-Hat transform increased detection rates by 15% by neutralizing the uneven overhead lighting of the operating room.

Future Improvements

  • Vision Transformers (ViT): To capture global texture context and ignore local fold patterns.
  • TensorRT Optimization: To enable real-time inference (30fps) on edge devices.

Applications

  • Sterile Processing Dept (SPD): Automated final check before packaging surgical kits.
  • Robotic Surgery: Real-time visual assistant for contaminant awareness.
